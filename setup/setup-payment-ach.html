Setup 3C

Create the ACH Payment Method

Create the ACH/bank payment method using the same setup pattern, with the required ACH values.

ACH payment method

For the ACH/bank payment method, follow the same steps used for the credit card payment method. Use these differences:

  • Payment Method ID: L2PACH
  • Means of Payment: Credit Card (This is intentional. Select Credit Card even for ACH payments.)
  • Description: linked2pay ACH/Bank Payments

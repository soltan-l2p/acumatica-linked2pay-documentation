Implementation Documentation

A complete guide to receiving customer credit card and ACH payments with linked2pay inside Acumatica.

Product scope

Supported

Receive customer payments

Collect Accounts Receivable payments by credit card or ACH, use saved tokens, apply processing fees, run AutoPayBill, synchronize customers and payment methods, and track gateway status in Acumatica.

Planned

Vendor payments

Vendor-payment processing is not available in the current release. It is planned for a future version and must not be included in current implementation scope, demonstrations, or testing.

Current scope: implement customer credit card and ACH collections only. Vendor payments are roadmap functionality, not a supported current workflow.

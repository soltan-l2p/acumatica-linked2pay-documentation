Setup 1A

Import the linked2pay Integration Package

Import the linked2pay customization package into the validated Acumatica sandbox.

Purpose

Load the approved linked2pay customization package without publishing it, so the contents and dependencies can be reviewed first.

Prerequisites

  • Completed Before You Begin.
  • Approved package ZIP and recorded checksum/version.
  • Customization Project import rights and current backup.
  • Sandbox on validated Acumatica build.

Acumatica navigation path and screen ID

Navigation: More Items > Customization > Customization Projects (SM204505)

Step-by-step instructions

Request the integration package

Import the integration package first.

Request the linked2pay integration package using the link below: Request Integration Package (.zip file)

In Acumatica, go to More Items > Customization > Customization Projects (Screen ID: SM204505).

Click the Import button at the top of the screen, then select the downloaded .zip file to import the customization project.

After the upload completes, the linked2payintegration project should appear under Customization Projects.

Open the linked2payintegration project and review the customization project details. The screen should look similar to the screenshot.

Expected result

The linked2payintegration project is visible in Customization Projects, opens successfully, and has not yet changed runtime screens.

Validation steps

  • Project opens without missing-file errors.
  • Package version/checksum matches the deployment record.
  • No unrelated customization is selected for publication.

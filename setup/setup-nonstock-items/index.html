Setup 4

Configure the Credit Card Fee Item

Create the CCFEE non-stock item used when linked2pay applies a card-processing fee.

Purpose

Provide the posting-controlled line item required by Apply Fee to create, release, and apply an AR Debit Memo for the processing fee.

Prerequisites

  • Card fees approved legally and operationally for the customer.
  • L2PCC FEE configured with approved percentage.
  • Posting class, sales account/subaccount, tax category, and UOM approved by accounting.
  • Non-Stock Items maintenance/release rights.

Acumatica navigation path and screen ID

Navigation: Receivables > Profiles > Non-Stock Items (IN202000)

Step-by-step instructions

In Acumatica, go to Receivables > Profiles > Non-Stock Items.

Create a new record for the non-stock item. Click the + button to create a new record.

Make sure the values match the following.

Click on Save to save the non-stock item.

The setup steps are complete. The extension is published, the processing centers are ready, and Acumatica has the payment methods needed for the linked2pay workflow.

Expected result

Apply Fee calculates the approved percentage, increases the payment total, creates and releases an AR Debit Memo with one CCFEE line, and applies that memo to the payment with the expected GL/tax result.

Validation steps

  • CCFEE can be selected/opened and is active.
  • Test fee equals base amount multiplied by FEE percent.
  • Debit memo description references the payment.
  • Debit memo is released/applied once.
  • GL accounts/subaccounts/period and tax are approved.
  • Removing fee before processing behaves as expected; removal after processing is blocked.

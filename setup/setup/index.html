Installation Path

Install and configure linked2pay in a controlled sequence with a validation gate after every step.

Ordered installation path

  1. Import and publish. Validate the package and plugin.
  2. Create card and ACH centers. Separate rails and payment types.
  3. Configure API and webhook. Test both paths.
  4. Create/link payment methods. Use L2PCC and L2PACH.
  5. Configure CCFEE. Only if card fees are in scope.
  6. Run formal testing. Credential success alone is not go-live approval.

Setup 3B

Link the Card Processing Center and AR Profile Field

Associate L2PCC with the card payment method and define the token field used by customer payment profiles.

Purpose

Link the card method to its processing center and add the required AR Payment Profile ID detail so Acumatica can store the linked2pay token reference.

Prerequisites

  • L2PCC processing center tested.
  • L2PCC payment method saved.
  • Approved card cash account linked.
  • Payment Method configuration rights.

Acumatica navigation path and screen ID

Navigation: Banking > Preferences > Payment Methods > L2PCC (CA204000)

Step-by-step instructions

Open the Processing Center tab and add the linked2pay processing center to the payment method. This processing center will handle credit card payments.

Click Search, select L2PCC, and then click Select. This is the credit card processing center you created earlier.

The payment method should now show the linked processing center.

Finally, configure "SETTINGS FOR USE IN AR". Add a new Payment Method Details row with these values:

  • ID: CCPID
  • Description: Payment Profile ID
  • Required: Checked
  • Payment Profile ID: Checked
  • Sort Order: 1
Click Save at the top of the page. The credit card payment method is now ready. Next, create the ACH payment method.

Expected result

L2PCC is assigned to the card payment method and new customer payment profiles expose a required CCPID token field.

Validation steps

  • Processing Center tab shows L2PCC after refresh.
  • Settings for Use in AR contains one CCPID row.
  • New customer payment method requires Payment Profile ID.
  • Test token resolves to the intended merchant profile.

linked2pay Extension

Clients will receive a ZIP package containing the required linked2pay extension files. Acumatica uses Extension Packages, allowing administrators to import the ZIP file and publish the extension without modifying the core application.

Once the extension has been published, the linked2pay payment processing components become available within Acumatica and are ready to be configured through the Processing Center.

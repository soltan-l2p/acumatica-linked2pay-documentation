Setup 2C

Configure the Webhook Connection

Create the Acumatica webhook, register it in linked2pay, and copy the webhook key back into Acumatica.

Open a new browser tab and go to Acumatica. Then go to More Items. Under Configuration, click Integration.

In the Integration section, under Preferences, click Webhooks. This opens the Webhooks page (Screen ID: SM304000).

Use linked2pay as the webhook name. For the implementation class, click Search and select Linked2PayIntegration.Webhooks.L2PWebhookHandler. Then click Save.

After the webhook is saved, Acumatica displays a URL field. Copy that URL, then return to the linked2pay portal.

In the linked2pay portal, return to API Settings. Under Webhooks, click Add Webhook.

Paste the Acumatica webhook URL into Payload URL. Copy the API Key; you will paste it into Acumatica in the next step.

Make sure your host is accessible so linked2pay can send the webhook. If your Acumatica instance is running on localhost or is not exposed to the public internet, the webhook may not work; during testing, you can use a public tunnel such as ngrok.

Make sure both ACH and Credit Card are selected, then click Save.

Important: If the linked2pay webhook API Key changes, update the key in every linked2pay processing center in Acumatica. Otherwise, Acumatica will not receive payment status updates through the webhook connection.

In Acumatica, paste the API Key from the linked2pay webhook into the WEBHOOKKEY field.

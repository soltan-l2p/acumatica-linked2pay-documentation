Setup Guide

Initial Setup

Before using the integration, the Linked2Pay processing center and payment methods must be configured in Acumatica.

Guided Walkthrough

Walk through setup one step at a time.

This page should feel like you are sitting beside the user during configuration. Start with payment methods, then move into the processing center and credential test.

Keep the screenshot close to the steps so the reader has a visual cue before they start clicking through Acumatica.

1

Create payment methods

Create or confirm payment methods such as L2PCC for credit card and L2PACH for ACH.

2

Create the processing center

Set up the Linked2Pay processing center and enter the required API settings.

3

Attach payment methods

Add L2PCC and L2PACH to the processing center payment methods tab and make them active.

4

Test credentials

Use the Test Credentials action to confirm Acumatica can connect to Linked2Pay successfully.

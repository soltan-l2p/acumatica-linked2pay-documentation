Setup Guide

Initial Setup

Follow these smaller pages in order to install the extension, configure the Linked2Pay processing centers, and prepare the payment methods.

How to Use This Section

Each page covers one focused part of the setup.

The original three setup pages are now divided into shorter walkthroughs. The screenshots stay in the same order, but each page is easier to read, review, and update later.

Start with importing the customization project, then configure the processing centers, and finish by confirming the payment methods.

1A Import Extension Download and import the Linked2Pay customization project. 1B Publish Extension Publish the customization project and confirm it completes successfully. 2A Create Center Create the Linked2Pay credit card processing center. 2B API Settings Copy the Linked2Pay API values into Acumatica. 2C Webhook Setup Connect Acumatica and Linked2Pay webhook settings. 2D ACH Center Save, test, and create the ACH processing center. 3A Credit Card Method Create and configure the credit card payment method. 3B Link Processing Center Attach the processing center and AR payment profile detail. 3C ACH Method Create the ACH payment method using the required differences.
Final result: the Linked2Pay extension is published, the processing centers are connected, and the payment methods are ready for the next workflow.

Customers

First Customer Sync

Use one action to create the customer in linked2pay and connect the Acumatica record.

Open the linked2pay Integration tab

Save the customer in Acumatica, then open the linked2pay Integration tab on Customers (AR303000). For a new customer, the Contact Profile ID is empty and read-only.

Click Sync with linked2pay

The same Sync action handles first-time creation and later updates. Because this customer has no Contact Profile ID, the integration creates a contact in linked2pay.

Confirm the connection

After a successful sync, Boarded to external gateway is checked and the read-only L2P Contact Profile ID is populated. Keep this identifier unchanged; it links the two records.

Choose the Auto-Bill policy

Enable Auto Bill for customers who should be charged automatically. For new customers, this field defaults to enabled only when Auto-Bill is enabled on both the L2PCC and L2PACH processing centers.

Verify linked2pay

Open My Contacts in the linked2pay portal and confirm that the new contact contains the customer information from Acumatica.

Data quality: the integration safely normalizes missing gateway-required values and omits an invalid email, but the Acumatica customer record should still be corrected with accurate contact and address information.

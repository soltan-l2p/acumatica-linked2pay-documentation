Customers

Board your Customer from Acumatica to linked2pay

Show how the customer record is sent from Acumatica into linked2pay for the first time.

linked2pay Tab

In the customer record (ScreenId: AR303000), you should be able to see the linked2pay tab.

Click on the Sync with linked2pay button to send the customer record from Acumatica into linked2pay.

You should see a confirmation message and the Boarded to external gateway checkbox should be checked.

And finally your linked2pay Contact Profile ID should be populated with the linked2pay contact profile ID.

Important

Make sure you enable Auto Bill by checking the Auto Bill checkbox if you want to use the AutoPayBill feature for this customer.

Recommended: If you don't want to do this manually for each customer, check Enable AutoPayBill for more details on how to enable AutoPayBill for newly customers automatically.

In linked2pay portal, you should be able to see the customer profile that was just created from Acumatica.

Click on it.

As you can see in the screenshot, the customer profile in linked2pay has the same information as the customer record in Acumatica. The handoff is complete.

At the end of this step, you should have a customer record in linked2pay that was sent from Acumatica.

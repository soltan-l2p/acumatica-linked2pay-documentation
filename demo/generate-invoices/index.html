Recurring Payments

Generate Invoices

Generate invoices from the recurring schedule and review the created documents.

Configure generation first

Set the execution date and number execution.

Your recurring transaction should appear on the list if you chose its date or any date after that.

Either

  • Check the specific recurring transaction and click on Run
  • Or simply just click on Run All if you want to run it for all the recurring transactions you have for the date you set.

You should be able to see result screen with different statuses.

To double check, click on Schedule ID to review the documents.

Result

Open Generated Documents tab to see newly generated invoices.

As you can see, Invoices have been generated based on given instructions (every 2 days).

Since you generated recurring transactions, let's use AutoBill to pay them out.

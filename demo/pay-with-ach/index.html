Invoice

Pay with ACH

Process a customer bank-account profile without a credit-card fee.

Select the linked2pay ACH profile

Start from a released invoice, click Pay, and select the customer's active linked2pay ACH profile and approved ACH cash account.

Confirm there is no card fee

L2PACH uses FEE=0. If an unprocessed payment was changed from a linked2pay card to ACH, the integration removes the card fee and its unprocessed fee memo before payment processing.

Process and monitor settlement

Run the payment action and verify the transaction in linked2pay. An accepted ACH request is not the same as final bank settlement; authenticated webhook updates keep the local transaction status aligned with later gateway events.

The current integration records external ACH status updates. Operational handling of returns and any accounting documents required by company policy remain a controlled follow-up process.

Level 3 Processing

Level 3 Processing is currently in development and is expected to be available within the next couple of days. Once completed, the integration will enable Acumatica to submit enhanced transaction data to linked2pay during payment authorization.

The implementation leverages linked2pay's Level 3 Processing API together with the detailed invoice and line-item information already available in Acumatica. During payment processing, the integration will automatically collect the required Level 3 data and include it in the authorization request sent to linked2pay.

This will provide businesses with richer transaction data, improved reporting, and the potential for lower interchange fees on eligible commercial card transactions, all without requiring additional manual steps from the user.

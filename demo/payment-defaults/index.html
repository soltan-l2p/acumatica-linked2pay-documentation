linked2pay Center

Assign Cash Accounts and Defaults

Prepare one or many customers for payment processing from one central Acumatica screen.

Open linked2pay Center

Go to Receivables > Processes > linked2pay Center (LP102000). The customer grid supports selected-row processing and Process All.

Assign a cash account

Set Operation to Assign Cash Account. Choose linked2pay Credit Card or linked2pay ACH, select an active AR cash account mapped to that payment method, then process selected customers or all customers.

Set the default payment method

Set Operation to Set Default Payment Method, choose the linked2pay payment method, and process the required customers. An eligible current default is preserved; otherwise the integration selects an active profile consistently.

Review every row

The processing window records success or the specific error for each customer. One failed customer does not stop the remaining rows. Correct failed setup and rerun only those customers.

These operations update Acumatica configuration only. They do not create, delete, or change wallet methods in linked2pay.

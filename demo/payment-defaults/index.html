Payment Methods

Set Default Payments and Cash Account

For each payment method that is synced from linked2pay, you must select a cash account in Acumatica. Additionally, you should set a recommended default payment method to streamline the payment workflow.

Under the Payment Methods section, click on one of the synced linked2pay payment methods and then click on the "View Payment Method" button.

In the opened window, select the appropriate cash account for this payment method and save the changes.

IMPORTANT: Do this step for each payment method you synced from linked2pay.

Default method

Choose the recommended default.

Select one of the synced linked2pay payment methods and check the box to set it as the recommended default payment method. This will help streamline the payment process for your customers.

This page is a demonstration of how to set the recommended default payment method and cash account for each synced linked2pay payment method in Acumatica. Following these steps ensures that your payment workflow is efficient and that all transactions are properly recorded.

Questions & Answers

Here are the most common questions before starting the Acumatica setup.

1. How does the Acumatica payment integration work with linked2pay?

Acumatica holds the ERP-side setup while linked2pay supports the payment connection, processing center configuration, and payment method workflow.

2. Does the Acumatica integration require setup?

Yes. The administrator publishes the customization, configures the processing centers, and prepares the payment methods before the workflow is used.

3. How does this help Acumatica users?

It gives users a clearer payment path inside Acumatica and keeps setup, screenshots, and handoff steps organized in one implementation flow.

4. Can users follow the setup step by step?

Yes. The setup guide is split into focused pages so each step covers one part of the Acumatica configuration.

5. Does the integration support multiple payment types?

Yes. The setup path covers credit card and ACH configuration so teams can prepare both payment methods.

6. Why does this overview page matter?

It gives the user the big picture before they begin, so the setup feels guided instead of becoming just a list of screens.

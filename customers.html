Customer Workflow

Boarding And Syncing Customers

A customer can be connected to Linked2Pay from the Acumatica Customer screen, then kept aligned through sync actions.

Guided Walkthrough

Make customer onboarding feel clear.

This page is about connecting one Acumatica customer to one Linked2Pay contact profile. Start with the customer record, then explain what boarding and sync each do.

Later, replace the placeholder with the actual Customer screen so the guide feels grounded in the user interface.

01

Board to gateway

Creates a Linked2Pay contact profile from the Acumatica customer. The returned Contact ID is saved on the customer record.

02

Sync with Linked2Pay

Reads the customer's Linked2Pay Contact ID, pulls current contact and payment profile data, and updates Acumatica.

Field Meaning
Linked2Pay Contact ID The customer or contact profile ID from Linked2Pay.
Boarded To Gateway Shows whether the customer has already been created or connected in Linked2Pay.
Enable Auto Bill Controls whether this customer is eligible for automated invoice payments.

Setup Step 2

Create the Linked2Pay Processing Center

Next, create the processing center that stores your Linked2Pay connection settings in Acumatica.

Now we connect the pieces

Let's set up the processing center.

In Acumatica, go to Banking > Credit Card Processing > Processing Centers (Screen ID: CA2050PL).

This is where Acumatica keeps the connection details for payment gateways. For Linked2Pay, this page becomes the main bridge between Acumatica and your Linked2Pay account.

Start by creating a new processing center. Click the + button to open the processing center setup form.

You should see a form like this. (Screen ID: CA205000).

1st Processing Center - For the Credit Card Payments

Please enter the following information into the processing center form:

  • Processing Center ID: L2PCC
  • Name: Linked2Pay Credit Card Payments
  • Active: Checked
  • Cash Account: Your Acumatica Cash Account

For the Payment Plug-In, click on Search icon and then pick Linked2PayIntegration.PaymentProcessor.

Once you pick Linked2PayIntegration.PaymentProcessor as the Payment Plug-In, you will see the following fields appear.

Linked2Pay Portal

Now we need to go to the Linked2Pay portal and configure the integration settings.

At the bottom of the sidebar, click on API Settings.

or simply use this link: Linked2Pay API Settings Page

So we are basically done creating the Linked2Pay processing center. This page is the connection point between Acumatica and Linked2Pay, and the next step is to confirm the payment methods that will use it.

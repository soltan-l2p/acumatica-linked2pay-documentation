Payments

With the integration in place, payments can be processed directly from Acumatica while seamlessly communicating with linked2pay.

In this demonstration, we'll create an invoice in Acumatica, process a payment, and see how the transaction is automatically mirrored in linked2pay. This keeps payment records synchronized across both systems.

We'll also demonstrate Auto Bill, showing how saved payment methods can be used to quickly and securely process customer payments without requiring payment details to be entered each time.

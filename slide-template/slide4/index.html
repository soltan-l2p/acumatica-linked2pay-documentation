linked2pay
Processing Centers

After installing the extension, the next step is configuring the linked2pay Processing Center. This is where Acumatica is provided with the credentials and connection settings needed to communicate with the linked2pay API.

During setup, you'll enter the required authentication details, such as your API credentials and other configuration options provided by linked2pay. Once saved, Acumatica can securely send payment requests and receive transaction responses through the linked2pay gateway.

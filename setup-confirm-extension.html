Setup Step 1

Confirm the Extension Is Installed

First, make sure the Linked2Pay customization project is published in Acumatica.

First of all

Let's install it first

Download the Linked2Pay extension from the link below: Download Extension (.zip file)

In Acumatica, go to More Items > Customization > Customization Projects (Screen ID: SM204505).

You should see the Import button at the top. Click it and select the downloaded .zip file to import the customization project.

Once you upload the extension, you should see the linked2payintegration project under Customization Projects.

Just to confirm, click on the linked2payintegration project and check the customization project. It should look similar to the screenshot on the side.

Let's Publish the Extension

Now go back to the Customization Projects screen (Screen ID: SM204505). Check the linked2payintegration project and click on the Publish button at the top. This will publish the customization project and make it available for use in Acumatica.

During the publishing process, you may see a warning from Acumatica that says, "Active website users will be logged out." This is built-in Acumatica behavior, and it is safe. Just be aware of the warning, then click Publish.

The publishing process may take several minutes to complete. Give it a few minutes to finish. When it is done, you should see these logs:

So we are basically done importing and publishing the linked2payintegration customization project. But that is not all; we still have to do some more configuration to make it work. So let's move on to the next step.

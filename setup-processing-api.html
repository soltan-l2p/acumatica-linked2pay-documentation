Setup 2B

Configure API Values

Copy the required values from Linked2Pay and enter them into the Acumatica processing center.

Linked2Pay Portal

Go to the Linked2Pay portal and open the API settings page. At the bottom of the sidebar, click API Settings.

You can also use this link: Linked2Pay API Settings Page

In the API Settings section, find the Web Service Key. Copy that value.

Acumatica

Return to Acumatica and paste the Web Service Key into the APIKEY field.

Important: This key allows Acumatica to communicate with Linked2Pay. If the Web Service Key changes in Linked2Pay, update it in Acumatica so the connection continues to work.

Recommended default

Set AUTOBILL to True if newly created customers should have Auto-Bill enabled by default. This value is used as the global default for the Auto-Bill Enabled field on new customer records.

Note: You can still manage this setting individually for each customer. This value only controls the default for newly created customers, so you can override it later when needed.

The FEE field controls the percentage of the credit card processing fee that is passed to customers. For example, enter 3 to add a 3% fee to invoices paid by credit card. Enter 0 if no fee should be added.

Enter your Linked2Pay username in the L2PUSER field. This should be the same username used to sign in to the Linked2Pay portal.

For the first processing center, set PAYMETHOD to Credit Card. This processing center is used for credit card payments. You will create a second processing center for ACH payments later.
